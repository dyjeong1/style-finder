## 1. 목표
- 업로드 드롭존에 실제 드래그앤드롭 파일 선택 기능을 추가한다.

## 2. 기능 스펙
- `dragenter`, `dragover`, `dragleave`, `drop`를 처리한다.
- 드롭된 파일 중 첫 번째 이미지 파일만 선택 상태로 반영한다.
- 드래그 활성 중 드롭존 스타일을 강조한다.
- 기존 클릭 업로드와 동일하게 미리보기/분석 흐름을 사용한다.

## 3. 비기능 스펙
- 모바일/터치 환경에서도 클릭 업로드는 그대로 유지되어야 한다.
- 이미지가 아닌 파일 드롭 시 사용자에게 기존 에러 메시지 흐름으로 안내한다.

## 4. 검증 스펙
- `cd frontend && npm run build`
- 이미지 파일 드롭 시 파일명이 반영되는지 확인
- 드래그 중 활성 스타일이 보이는지 확인
