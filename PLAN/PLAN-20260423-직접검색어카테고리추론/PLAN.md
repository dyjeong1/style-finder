## 1. 배경/문제 정의
- 직접 검색어 `검은색 신발` 입력 시 전체 보기에서 `검은색 신발 상의`, `검은색 신발 하의` 같은 쿼리가 생성된다.
- 네이버 쇼핑은 `신발` 키워드를 강하게 반영해 상의/하의 섹션에도 신발 상품을 반환할 수 있다.

## 2. 목표/가설
- 1차 지표(Primary): 제품군이 명시된 직접 검색어는 해당 제품군 카테고리로만 조회한다.
- 2차 지표(Secondary): 제품군이 없는 직접 검색어는 기존처럼 전체 카테고리 분산 조회를 유지한다.
- 가설: 검색어 내 제품군을 추론하면 섹션과 상품군의 불일치를 줄일 수 있다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 직접 검색어 제품군 키워드 추론
  - 전체 보기 카테고리 쿼리 생성 범위 조정
  - 백엔드 테스트 보강
  - 문서 갱신
- 제외 범위:
  - 프론트 UI 변경
  - 네이버 검색 결과 후처리/재랭킹
