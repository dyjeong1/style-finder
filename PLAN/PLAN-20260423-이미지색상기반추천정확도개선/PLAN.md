## 1. 배경/문제 정의
- 현재 추천은 업로드 이미지의 실제 픽셀을 충분히 분석하지 않아 검색어로 잡힌 상품이 실제 코디 이미지와 덜 비슷하게 보일 수 있다.
- 특히 블랙/화이트/베이지/브라운/네이비처럼 쇼핑 탐색에서 중요한 색상 신호가 추천 검색어와 랭킹에 직접 반영되지 않았다.

## 2. 목표/가설
- 1차 지표(Primary): 업로드 이미지의 주요 색상 키워드를 추천 검색어에 반영한다.
- 2차 지표(Secondary): 추천 점수에 색상 일치 보너스를 추가한다.
- 가설: 기본 색상을 반영하면 같은 품목 안에서 이미지와 더 비슷한 상품이 상단에 노출된다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 업로드 이미지 실제 픽셀 기반 대표 색상 분석
  - 분석 결과에 `dominant_color` 필드 추가
  - 네이버 자동 검색어에 대표 색상 키워드 반영
  - 추천 점수 `score_breakdown.color_bonus` 추가
  - 프론트 분석 요약/매칭 정보에 색상 표시
  - 테스트/문서 갱신
- 제외 범위:
  - CLIP/딥러닝 이미지 임베딩 도입
  - 네이버 상품 이미지 다운로드 후 픽셀 분석

## 5. 리스크 & 가정
- Pillow가 없거나 이미지 파싱에 실패하면 기존 해시 기반 분석으로 fallback한다.
- 상품 자체 이미지를 분석하지 않는 한 완전한 시각 유사도는 제한적이다.
- 이번 개선은 현재 네이버 쇼핑 검색 구조 안에서 체감 정확도를 높이는 1차 개선이다.
