## 1. 목표
- 우측 상단 `Local Mode` 노출을 제거한다.
- 업로드 안내 문구를 요청한 문장으로 교체한다.
- 허용 이미지 형식을 화면에서 바로 확인할 수 있게 노출한다.

## 2. 기능 스펙
- 헤더 우측 상단에서 `Local Mode` 배지는 렌더링하지 않는다.
- 기존 `Reset Upload` 버튼 동작은 유지한다.
- 업로드 페이지 상단 보조 문구를 `이미지를 넣으면 유사한 상품을 추천해드립니다.`로 변경한다.
- 업로드 박스 안에 `PNG, JPG, JPEG, WEBP` 허용 형식 문구를 추가한다.

## 3. 비기능 스펙
- 헤더 전체 레이아웃이 급격히 무너지지 않도록 기존 메뉴/버튼 구조를 유지한다.
- 빌드 오류 없이 정적 페이지 생성이 가능해야 한다.

## 4. 검증 스펙
- `cd frontend && npm run build`
- `/upload`에서 헤더 상태 배지 제거 및 문구 변경 확인
