## 1. 목표
- 업로드 화면을 큰 인트로 패널, 독립된 업로드 드롭존, 우측 최근 업로드 패널 구조로 리디자인한다.

## 2. 기능 스펙
- 좌측 상단: 인트로 카피 + 큰 미리보기/무드 패널
- 좌측 하단: 업로드 드롭존 + 메인 CTA 버튼
- 우측: 최근 업로드 카드 리스트 + 아이템 수 배지
- 기존 파일 선택/업로드/추천 이동/최근 업로드 재사용 기능은 유지한다.

## 3. 비기능 스펙
- 데스크톱과 모바일 모두 레이아웃이 깨지지 않아야 한다.
- 기존 Pretendard/웜 톤 스타일과 조화를 유지한다.

## 4. 검증 스펙
- `cd frontend && npm run build`
- `/upload`에서 파일 선택, 업로드 버튼, 최근 업로드 재사용 버튼 확인
