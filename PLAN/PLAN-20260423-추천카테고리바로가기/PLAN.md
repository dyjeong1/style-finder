## 1. 배경/문제 정의
- 추천 결과가 카테고리별 섹션으로 나뉘었지만, 화면 상단에서 어떤 카테고리에 몇 개의 상품이 있는지 바로 알기 어렵다.
- 사용자는 관심 카테고리로 빠르게 이동하고 싶다.

## 2. 목표/가설
- 1차 지표(Primary): 전체 추천 결과 상단에서 카테고리별 상품 개수와 최고 매칭률을 빠르게 확인한다.
- 2차 지표(Secondary): 클릭 시 해당 카테고리 섹션으로 이동한다.
- 가설: 카테고리 요약/바로가기 바를 추가하면 추천 페이지 탐색성이 좋아진다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 전체 추천 보기에서 카테고리 요약 바 표시
  - 카테고리 요약 클릭 시 해당 섹션으로 이동
  - 카테고리별 상품 개수/최고 매칭률 표시
  - 문서/검증 갱신
- 제외 범위:
  - 백엔드 API 변경
  - 추천 정렬 로직 변경
