## 1. 배경/문제 정의
- 현재 추천 검색어는 카테고리 단위까지는 반영되지만 품목명이 충분히 세분화되지 않는다.
- 고정 영역 기반 감지는 겹치는 아이템과 없는 카테고리를 완전히 안정적으로 분리하지 못한다.
- 색상/텍스트 중심 점수만으로는 실제 이미지 유사도를 끝까지 끌어올리기 어렵다.

## 2. 목표/가설
- 1차 지표(Primary): 카테고리별 검색어가 `셔츠`, `데님 팬츠`, `메리제인 슈즈`, `숄더백`, `안경`처럼 더 구체적인 품목명을 포함한다.
- 2차 지표(Secondary): 없는 카테고리 오탐을 줄이고, 선택적 비전 임베딩 재정렬 시 랭킹 품질을 높인다.
- 가설: 연결 컴포넌트 기반 분리와 품목명 규칙, 비전 임베딩 재정렬을 결합하면 추천 품질과 설명력이 함께 좋아진다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 카테고리별 세부 품목명 추출
  - 연결 컴포넌트 기반 다중 아이템 분리
  - 선택적 CLIP 임베딩 재정렬 서비스/설정/테스트
  - 업로드 분석 응답/프론트 표시 개선
- 제외 범위:
  - 대규모 객체 탐지 모델 학습
  - 외부 유료 비전 API 의존

- CLIP는 로컬 패키지 설치 여부에 따라 선택적으로만 동작한다.
- heuristic와 비전 임베딩을 혼합하므로 초기 가중치 튜닝이 필요할 수 있다.
