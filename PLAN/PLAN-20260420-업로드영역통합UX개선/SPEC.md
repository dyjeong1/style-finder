## 1. 목표
- 업로드 선택 영역과 미리보기 영역을 하나로 합치고, 선택/삭제 흐름을 직관적으로 만든다.

## 2. 기능 스펙
- 통합 업로드 영역 클릭 시 파일 선택창이 열린다.
- 이미지 선택 후 같은 영역 안에 정사각형 미리보기가 표시된다.
- 삭제 버튼 클릭 시 선택 파일, 미리보기, 에러/성공 상태가 초기화된다.
- `이미지 분석하기` 버튼은 선택 파일이 있을 때만 활성화된다.

## 3. 비기능 스펙
- 드래그앤드롭 지원은 유지한다.
- 정사각형 미리보기는 데스크톱/모바일 모두에서 안정적으로 보인다.

## 4. 검증 스펙
- `cd frontend && npm run build`
- `/upload`에서 선택 → 미리보기 → 삭제 흐름 수동 확인
