## 목표
직접 검색어의 하위 품목명을 상의/하의/아우터/신발/가방 카테고리로 더 정확히 추론한다.

## 백엔드 스펙
- `CATEGORY_KEYWORDS`의 모든 카테고리에 대표 하위 품목명을 추가한다.
- 상의 예: 티셔츠, 셔츠, 니트, 블라우스, 민소매, 크롭티, 폴로, 베스트
- 하의 예: 슬랙스, 청바지, 와이드팬츠, 조거팬츠, 쇼츠, 미니스커트, 레깅스
- 아우터 예: 트렌치코트, 블레이저, 패딩, 바람막이, 무스탕, 플리스, 야상
- 가방 예: 호보백, 버킷백, 클러치, 에코백, 메신저백, 체인백, 보스턴백
- 신발은 기존 확장 사전을 유지하고 필요한 대표 품목을 보강한다.

## 검증
- `cd backend && PYTHONPATH=. python3 -m pytest tests -q` 성공
