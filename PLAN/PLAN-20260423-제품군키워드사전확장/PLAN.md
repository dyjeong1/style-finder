## 1. 배경/문제 정의
- 직접 검색어의 제품군 추론은 키워드 사전에 의존한다.
- 신발 외 카테고리의 하위 품목 키워드가 부족하면 `슬랙스`, `크롭 니트`, `트렌치코트`, `호보백` 같은 검색어가 전체 카테고리로 퍼질 수 있다.

## 2. 목표/가설
- 1차 지표(Primary): 대표 하위 품목 검색어가 올바른 카테고리로 추론된다.
- 2차 지표(Secondary): 제품군 없는 검색어는 기존처럼 전체 카테고리 분산 조회를 유지한다.
- 가설: 카테고리별 품목 사전을 확장하면 직접 검색어 기반 추천의 카테고리 오염이 줄어든다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 상의/하의/아우터/신발/가방 키워드 사전 확장
  - 카테고리별 대표 검색어 추론 테스트 추가
  - 문서 갱신
- 제외 범위:
  - 프론트 UI 변경
  - 네이버 검색 결과 후처리 필터
  - 외부 품목 분류 모델 도입

## 5. 리스크 & 가정
- 품목 사전은 완전한 자연어 이해가 아니라 검색어 의도 보정을 위한 실용 사전이다.
- 여러 카테고리 품목이 함께 들어간 검색어는 해당 카테고리들을 모두 조회한다.
