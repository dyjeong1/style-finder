## 1. 배경/문제 정의
- 이미지 분석 기반 자동 검색어만으로는 사용자가 원하는 상품 맥락을 정확히 반영하기 어려울 수 있다.
- 로컬 개인 도구에서는 사용자가 직접 검색어를 보정할 수 있는 흐름이 필요하다.

## 2. 목표/가설
- 1차 지표(Primary): 추천 페이지에서 직접 입력한 검색어로 추천 결과를 다시 조회할 수 있다.
- 2차 지표(Secondary): 전체 보기에서는 입력 검색어에 카테고리 키워드를 붙여 다양한 제품군 섹션을 유지한다.
- 가설: 직접 검색어 입력을 제공하면 네이버 쇼핑 추천 결과를 사용자가 빠르게 원하는 방향으로 튜닝할 수 있다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 백엔드 `/recommendations`에 `custom_query` 파라미터 추가
  - 직접 입력 검색어 기반 네이버 쇼핑 쿼리 생성
  - 프론트 추천 페이지 검색어 입력/적용/초기화 UI 추가
  - 테스트/문서 갱신
- 제외 범위:
  - 검색어 저장 히스토리
  - 검색어 자동 추천
