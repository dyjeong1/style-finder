## 1. 배경/문제 정의
- 네이버 쇼핑 연동 후 추천 상품은 실제 데이터로 표시되지만, 카드 안에 분석 점수와 메타 정보가 과도하게 노출된다.
- 사용자는 추천 결과에서 상품 이미지, 상품명, 가격, 구매/찜 액션을 우선 보고 싶다.

## 2. 목표/가설
- 1차 지표(Primary): 추천 카드에서 상품 이미지/상품명/가격/액션의 시각적 우선순위가 명확해진다.
- 2차 지표(Secondary): 유사도/분석값은 유지하되 쇼핑 경험을 방해하지 않도록 보조 정보화한다.
- 가설: 점수 정보는 접고 쇼핑 액션 중심으로 정리하면 추천 페이지의 체감 완성도가 높아진다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 추천 카드 마크업 재구성
  - 상품 이미지/배지/가격/버튼 스타일 개선
  - 유사도 및 분석 신호를 보조 정보 영역으로 정리
  - 모바일 반응형 유지
  - 문서/검증 갱신
- 제외 범위:
  - 추천 API 스키마 변경
  - 상품 상세 페이지 추가
