## 1. 목표
- 업로드 화면에서 핵심 행동과 정보만 남기도록 문구와 버튼을 줄인다.

## 2. 기능 스펙
- 제거 대상: 영어 eyebrow/배지, 보조 안내 문구, 최근 업로드 개수 배지, 최근 업로드 CTA 버튼
- 최근 업로드는 카드 전체를 클릭 가능한 형태로 바꿔 재사용 기능 유지
- 업로드 드롭존 문구와 헤더 카피는 더 짧은 한국어로 정리

## 3. 비기능 스펙
- 시각적 여백은 유지하되 텍스트 밀도를 줄인다.
- 접근성을 위해 카드 클릭은 `button` 또는 명확한 상호작용 요소로 유지한다.

## 4. 검증 스펙
- `cd frontend && npm run build`
- `/upload`에서 최근 업로드 카드 클릭 시 추천 페이지 이동 확인
