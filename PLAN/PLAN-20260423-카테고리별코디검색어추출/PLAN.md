## 1. 배경/문제 정의
- 테스트 이미지처럼 배경이 베이지인 경우 전체 평균색이 베이지로 잡혀 모든 카테고리 검색어가 베이지로 생성된다.
- 실제 추천은 흰색 바지, 아이보리 숄더백, 흰색 셔츠, 검은색 조끼, 갈색 메리제인슈즈처럼 품목별 색상/형태가 분리되어야 한다.

## 2. 목표/가설
- 1차 지표(Primary): 카테고리별 검색어가 전체 대표색이 아니라 해당 영역의 품목/색상 힌트를 사용한다.
- 2차 지표(Secondary): 배경색이 검색어에 과도하게 반영되지 않는다.
- 가설: 배경색 제외와 영역별 색상 힌트를 적용하면 네이버 검색 후보가 실제 코디 구성품에 가까워진다.

## 3. 범위/산출물(Scope & Deliverables)
- 포함 범위:
  - 이미지 edge 기반 배경색 추정
  - 영역별 foreground 색상 추출
  - 상의/하의/아우터/신발/가방 검색어 힌트 생성
  - 네이버 자동 검색어에 카테고리별 힌트 우선 반영
  - 테스트/문서 갱신
- 제외 범위:
  - 딥러닝 객체 탐지
  - CLIP 임베딩
