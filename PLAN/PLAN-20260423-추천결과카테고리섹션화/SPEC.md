## 목표
추천 결과 전체 보기에서 상품을 카테고리별 섹션으로 나누어 다양한 제품군이 잘 드러나도록 한다.

## 프론트 스펙
- `category` 필터가 비어 있고 추천 결과가 있으면 카테고리 섹션 리스트를 표시한다.
- 섹션 순서는 `상의`, `하의`, `아우터`, `신발`, `가방`, `기타` 순으로 표시한다.
- 각 섹션 헤더에는 카테고리명, 상품 개수, 최고 매칭률을 표시한다.
- 특정 카테고리 필터가 선택되면 기존 단일 카드 그리드를 유지한다.
- 기존 상품 카드의 상품 보기/위시리스트/매칭 정보 동작은 유지한다.
- 모든 폰트 크기는 기존 규칙대로 `pt` 단위를 사용한다.

## 검증
- `cd frontend && npm run build` 성공
