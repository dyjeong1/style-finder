## 1. 목표
- 업로드 전에는 미리보기 정사각형 영역을 숨긴다.
- 파일 선택 후에만 정사각형 이미지 미리보기를 노출한다.

## 2. 기능 스펙
- 업로드 박스는 파일 미선택 상태에서 안내 텍스트만 렌더링한다.
- 파일 선택 후 `filePreviewUrl`이 생기면 정사각형 미리보기 박스를 렌더링한다.
- `사진 삭제`를 누르면 미리보기와 선택 파일이 함께 초기화된다.
- 허용 형식 문구는 미리보기가 없어도 업로드 박스 안에서 계속 보여야 한다.

## 3. 비기능 스펙
- 업로드 박스 높이가 너무 비거나 과도하게 커 보이지 않도록 상태별 간격을 보정한다.
- 빌드 오류 없이 정적 페이지 생성이 가능해야 한다.

## 4. 검증 스펙
- `cd frontend && npm run build`
- `/upload`에서 파일 선택 전/후 미리보기 노출 차이 확인
